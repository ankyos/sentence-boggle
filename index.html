How to Play

One device = one local team session. It saves only on this iPad/browser.

Class roles

  • Finder: points to possible paths.
  • Assembler: taps the path, but stays silent.
  • Coach: says if it sounds natural.
  • Speaker: uses it in the finale.

Scoring

  • Valid sentence: 3 pts
  • Longer connected path: +1 each tile after 3
  • Mission goal hit: +2
  • Because/but/so: +2
  • Peer vote: +3

Teacher note

This is intentionally “Boggle-ish,” not pure Boggle. Tap only touching tiles, and follow the colour pattern: Who → Verb → Thing. When three tiles could make a sentence, pick the natural option from the modal.
